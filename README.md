# 📖 [TDD Project] <a name="about-project"></a>

## 💻 Getting Started <a name="getting-started"></a>

### Preriquisites

Ruby version >= 2.7.0

### Setup

Clone this repository to your desired folder:
sh
  cd my-folder
 git clone https://github.com/codedit334/Test_Driven_Development_Project.git

 ### Install:
 Run: bundle install

 ### Start:
 Run: ruby main.rb
